Surfer#

class ansys.meshing.prime.autogen.surfer.Surfer(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager, part_id: int)#

Generates surface mesh.

Performs surface meshing using various surface meshing algorithms on topofaces or face zonelets. For example, constant size or volumetric size surface meshing.

Parameters:

  • model (Model) – Server model to create Surfer object.

  • part_id (int) – Id of the part.

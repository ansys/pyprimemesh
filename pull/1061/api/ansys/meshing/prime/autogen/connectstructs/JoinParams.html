JoinParams#

class ansys.meshing.prime.autogen.connectstructs.JoinParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, tolerance: float = None, use_absolute_tolerance: bool = None, remesh: bool = None, match_angle: float = None, overlap_zone_name: str = None, json_data: dict = None, **kwargs)#

Parameters used for join.

Parameters:

  • model (Model) – Model to create a JoinParams object with default parameters.

  • tolerance (float, optional) – Overlap tolerance between overlapping zonelets.

  • use_absolute_tolerance (bool, optional) – Tolerance provided is absolute value.

  • remesh (bool, optional) – Remesh at overlap surface boundary.

  • match_angle (float, optional) – Match angle determines face pair inclination for overlap consideration.

  • overlap_zone_name (str, optional) – Zone id to be assigned to overlap zonelets belonging to different zones.

  • json_data (dict, optional) – JSON dictionary to create a JoinParams object with provided parameters.

Examples

>>> join_params = prime.JoinParams(model = model)

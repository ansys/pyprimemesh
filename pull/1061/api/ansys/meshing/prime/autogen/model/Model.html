Model#

class ansys.meshing.prime.autogen.model.Model(comm, id: int, object_id: int, name: str)#

Bases: ansys.meshing.prime.autogen.coreobject.CoreObject, ansys.meshing.prime.autogen.coreobject.CommunicationManager

Model is the nucleus of Prime. Model forms the base and contains all the information about Prime.

You can access any information in Prime only through Model. Model allows you to query TopoData, ControlData, Parts, SizeFields and more.

Parameters:

  • comm (Communicator) – Communicator to connect with the Ansys Prime server.

  • id (int) – Id of the Model.

  • object_id (int) – Object id of the Model.

  • name (str) – Name of the Model.

SurfaceUtilities#

class ansys.meshing.prime.autogen.surfaceutilities.SurfaceUtilities(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager)#

Bases: ansys.meshing.prime.autogen.coreobject.CoreObject

Performs various general surface utilities algorithms. For example, copy zonelets, resolve surface intersections.

Parameters:

model (Model) – Server model to create SurfaceUtilities object.

Model#

class ansys.meshing.prime.core.model.Model(comm: ansys.meshing.prime.internals.communicator.Communicator, id: int, object_id: int, name: str)#

Contains all information about Ansys Prime Server.

This class provides the nucleus of Ansys meshing technology. You can access any information in Ansys Prime Server only through the Model class. Using this class, you can query topo data, control data, parts, size fields, and more.

Parameters:

  • comm (Communicator) – Communicator to connect with the Ansys Prime server.

  • id (int) – ID of the model.

  • object_id (int) – Object ID of the model.

  • name (str) – Name of the model.

SizeControl#

class ansys.meshing.prime.core.sizecontrol.SizeControl(model, id, object_id, name, local=False)#

Size control is used to compute the size field.

The size field is computed based on the size control defined. Different type of size controls provide control over how the mesh size is distributed on a surface or within the volume.

Parameters:

  • model (Model) – Server model to create SizeControl object.

  • id (int) – Id of the SizeControl.

  • object_id (int) – Object id of the SizeControl.

  • name (str) – Name of the SizeControl..

  • local (bool, optional) – Unused. The default is False.

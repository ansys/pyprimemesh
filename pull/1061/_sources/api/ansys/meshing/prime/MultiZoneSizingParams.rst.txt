   Parameters for MultiZone meshing.

   :param model: Model to create a ``MultiZoneSizingParams`` object with default parameters.
   :type model: Model
   :param max_size: Defines global maximum mesh size.

                    **This is a beta parameter**. **The behavior and name may change in the future**.
   :type max_size: float, optional
   :param min_size: Defines global minimum mesh size.

                    **This is a beta parameter**. **The behavior and name may change in the future**.
   :type min_size: float, optional
   :param growth_rate: Defines growth rate.

                       **This is a beta parameter**. **The behavior and name may change in the future**.
   :type growth_rate: float, optional
   :param use_volumetric_size_field: Defines whether to use size field for MultiZone meshing.

                                     **This is a beta parameter**. **The behavior and name may change in the future**.
   :type use_volumetric_size_field: bool, optional
   :param json_data: JSON dictionary to create a ``MultiZoneSizingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> multi_zone_sizing_params = prime.MultiZoneSizingParams(model = model)

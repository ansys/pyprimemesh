   Parameters to control part summary results.

   :param model: Model to create a ``PartSummaryParams`` object with default parameters.
   :type model: Model
   :param print_id: Boolean to control print ids. The default is false.
   :type print_id: bool, optional
   :param print_mesh: Boolean to control print mesh information. The default is true.
   :type print_mesh: bool, optional
   :param json_data: JSON dictionary to create a ``PartSummaryParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> part_summary_params = prime.PartSummaryParams(model = model)

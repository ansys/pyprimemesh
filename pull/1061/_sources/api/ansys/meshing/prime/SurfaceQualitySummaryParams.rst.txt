   Parameters to control surface quality summary results.

   :param model: Model to create a ``SurfaceQualitySummaryParams`` object with default parameters.
   :type model: Model
   :param face_quality_measures: List of face quality measures for surface quality diagnostics.
   :type face_quality_measures: List[FaceQualityMeasure], optional
   :param scope: Scope the face zonelets for surface quality diagnostics.
   :type scope: ScopeDefinition, optional
   :param quality_limit: Quality limit per face quality measure. If the quality limit is not specified, the default quality limit is used.
   :type quality_limit: Iterable[float], optional
   :param json_data: JSON dictionary to create a ``SurfaceQualitySummaryParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> surface_quality_summary_params = prime.SurfaceQualitySummaryParams(model = model)

:class:`MeshStackerParams`
==========================


.. py:class:: ansys.meshing.prime.MeshStackerParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, origin: ansys.meshing.prime.params.primestructs.Iterable[float] = None, direction: ansys.meshing.prime.params.primestructs.Iterable[float] = None, lateral_defeature_tolerance: float = None, stacking_defeature_tolerance: float = None, max_offset_size: float = None, size_control_ids: ansys.meshing.prime.params.primestructs.Iterable[int] = None, seed_faces: ansys.meshing.prime.params.primestructs.Iterable[int] = None, delete_base: bool = None, json_data: dict = None, **kwargs)



   Input parameters associated with the mesh stacker operations.

   :param model: Model to create a ``MeshStackerParams`` object with default parameters.
   :type model: Model
   :param origin: Origin coordinate list of stacker.
   :type origin: Iterable[float], optional
   :param direction: Direction vector of stacker.
   :type direction: Iterable[float], optional
   :param lateral_defeature_tolerance: Absolute lateral distance tolerance for stacker. If the lateral distance tolerance is not specified, a default tolerance value is calculated by stacker.
   :type lateral_defeature_tolerance: float, optional
   :param stacking_defeature_tolerance: Absolute stacking distance tolerance for stacker. If the stacking distance tolerance is not specified, a default tolerance value is calculated by stacker.
   :type stacking_defeature_tolerance: float, optional
   :param max_offset_size: Maximum stack size allowed during stacking. If the maximum stack size is not specified, it is set to global max size.
   :type max_offset_size: float, optional
   :param size_control_ids: List of size control ids to be respected by stacker. Stacker respects all supported controls by default.
   :type size_control_ids: Iterable[int], optional
   :param seed_faces: List of faces whose edges need to be imprinted on the base face. If the faces are meshed, the mesh will be transferred to the base face.

                      **This is a beta parameter**. **The behavior and name may change in the future**.
   :type seed_faces: Iterable[int], optional
   :param delete_base: Option to delete base face at the end of stacking. The default is false.
   :type delete_base: bool, optional
   :param json_data: JSON dictionary to create a ``MeshStackerParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> mesh_stacker_params = prime.MeshStackerParams(model = model)

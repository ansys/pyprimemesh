:class:`MorphBCParams`
======================


.. py:class:: ansys.meshing.prime.MorphBCParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, morph_region_method: BCsVolumetricModality = None, morphable_layers: int = None, morph_region_box_extension: float = None, json_data: dict = None, **kwargs)



   MorphBCParams contains the input parameters for calculating the boundary conditions for a morphing problem.

   :param model: Model to create a ``MorphBCParams`` object with default parameters.
   :type model: Model
   :param morph_region_method: Indicate options to identify morphable regions in the input volumetric mesh.
   :type morph_region_method: BCsVolumetricModality, optional
   :param morphable_layers: Number of layers around defined boundary.
   :type morphable_layers: int, optional
   :param morph_region_box_extension: Percentage extension for the volumetric box.
   :type morph_region_box_extension: float, optional
   :param json_data: JSON dictionary to create a ``MorphBCParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> morph_bcparams = prime.MorphBCParams(model = model)

.. py:function:: download_block_model_fmd(destination: Optional[str] = None, force: bool = False) -> Union[str, os.PathLike]

   Download CAD file for the block model example.

   :param destination: Path to download the example file to. The default
                       is ``None``, in which case the default path for app data
                       is used.
   :type destination: str, optional
   :param force: Whether to download the example file. The default is
                 ``False``, in which case if the example file is cached, it
                 is reused.
   :type force: bool, optional

   :returns: Local path to the downloaded file.
   :rtype: str

   .. rubric:: Examples

   >>> import ansys.meshing.prime as prime
   >>> import ansys.meshing.prime.examples as prime_examples
   >>> with prime.launch_prime() as session:
   >>>     model = session.model
   >>>     block_model = prime_examples.download_block_model_fmd()
   >>>     with prime.FileIO(model) as io:
   >>>         _ = io.import_cad(block_model, params=prime.ImportCADParams(model))
   >>>     print(model)

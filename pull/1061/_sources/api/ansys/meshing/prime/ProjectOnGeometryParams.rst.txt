:class:`ProjectOnGeometryParams`
================================


.. py:class:: ansys.meshing.prime.ProjectOnGeometryParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, project_only_mid_nodes: bool = None, project_on_facets_if_cadnot_found: bool = None, json_data: dict = None, **kwargs)



   Parameters used to project topoface nodes on associated geometry.

   :param model: Model to create a ``ProjectOnGeometryParams`` object with default parameters.
   :type model: Model
   :param project_only_mid_nodes: Option to project only the mid nodes.

                                  **This is a beta parameter**. **The behavior and name may change in the future**.
   :type project_only_mid_nodes: bool, optional
   :param project_on_facets_if_cadnot_found: Option to project on facet if geometry is not found.

                                             **This is a beta parameter**. **The behavior and name may change in the future**.
   :type project_on_facets_if_cadnot_found: bool, optional
   :param json_data: JSON dictionary to create a ``ProjectOnGeometryParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> project_on_geometry_params = prime.ProjectOnGeometryParams(model = model)







:class:`PrismControl`
=====================


.. py:class:: ansys.meshing.prime.core.prismcontrol.PrismControl(model, id, object_id, name, local=False)

   Bases: :py:obj:`ansys.meshing.prime.autogen.prismcontrol.PrismControl`


   PrismControl allows you to generate prisms.

   PrismControl allows you to control generation of prisms.
   Controls include setting the face scope, volume scope and growth parameters.

   :param model: Server model to create PrismControl object.
   :type model: Model
   :param id: Id of the PrismControl.
   :type id: int
   :param object_id: Object id of the PrismControl.
   :type object_id: int
   :param name: Name of the PrismControl.
   :type name: str

:class:`DatabaseProcessor`
==========================


.. py:class:: ansys.meshing.prime.core.dynaexportutils.DatabaseProcessor(model: ansys.meshing.prime.Model, steps_data: dict, card_format: str = 'SHORT')

   Processes the part's simulation data and generates LS-DYNA cards for database keywords.

   :param model: Model that the methods are to work on.
   :type model: prime.Model
   :param steps_data: The steps data extracted from the part's simulation data.
   :type steps_data: dict
   :param card_format: The LS-DYNA card format for writing. Defaults to "SHORT".
   :type card_format: str, optional

   .. rubric:: Notes

   **This is a beta API**. **The behavior and implementation may change in future**.

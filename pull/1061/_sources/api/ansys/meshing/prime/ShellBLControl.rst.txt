:class:`ShellBLControl`
=======================


.. py:class:: ansys.meshing.prime.ShellBLControl(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager, id: int, object_id: int, name: str)



   ShellBLControl allows you to generate quad mesh on face zonelets.

   ShellBLControl allows you to control generation of quad mesh on face zonelets. Controls include setting the edge scope, face scope and growth parameters.

   :param model: Server model to create ShellBLControl object.
   :type model: Model
   :param id: Id of the ShellBLControl.
   :type id: int
   :param object_id: Object id of the ShellBLControl.
   :type object_id: int
   :param name: Name of the ShellBLControl.
   :type name: str

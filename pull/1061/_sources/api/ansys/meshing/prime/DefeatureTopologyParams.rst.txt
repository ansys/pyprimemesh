:class:`DefeatureTopologyParams`

   :param model: Model to create a ``DefeatureTopologyParams`` object with default parameters.

   :type allow_curved_topo_faces: bool, optional

                                 **This is a beta parameter**. **The behavior and name may change in the future**.
   :type fillet_spanning_angle: float, optional
   :param aggressive_edge_merge: Indicate whether to allow aggressive edge merge while performing partial defeature.

                                 **This is a beta parameter**. **The behavior and name may change in the future**.
   :type aggressive_edge_merge: bool, optional
   :param thin_stripes_tolerance: Topoface width tolerance to detect thin faces below the provided value.

                                  **This is a beta parameter**. **The behavior and name may change in the future**.
   :type thin_stripes_tolerance: float, optional
   :param partial_defeature_params: Parameters for partial defeature operation.

                                    **This is a beta parameter**. **The behavior and name may change in the future**.
   :type partial_defeature_params: PartialDefeatureParams, optional
   :param delete_interior_nodes_params: Parameters for delete interior nodes operation.

                                        **This is a beta parameter**. **The behavior and name may change in the future**.
   :type delete_interior_nodes_params: DeleteInteriorNodesParams, optional
   :param json_data: JSON dictionary to create a ``DefeatureTopologyParams`` object with provided parameters.
   :type json_data: dict, optional

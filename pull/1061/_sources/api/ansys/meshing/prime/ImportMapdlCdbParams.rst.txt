:class:`ImportMapdlCdbParams`
=============================


.. py:class:: ansys.meshing.prime.ImportMapdlCdbParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, drop_mid_nodes: bool = None, append: bool = None, json_data: dict = None, **kwargs)



   Parameters to control MAPDL CDB import settings.

   :param model: Model to create a ``ImportMapdlCdbParams`` object with default parameters.
   :type model: Model
   :param drop_mid_nodes: Option to import quadratic mesh elements as linear by skipping mid nodes.
   :type drop_mid_nodes: bool, optional
   :param append: Option to append imported cdb into existing model.
   :type append: bool, optional
   :param json_data: JSON dictionary to create a ``ImportMapdlCdbParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> import_mapdl_cdb_params = prime.ImportMapdlCdbParams(model = model)

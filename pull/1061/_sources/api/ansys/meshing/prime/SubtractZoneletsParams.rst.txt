:class:`SubtractZoneletsParams`
===============================


.. py:class:: ansys.meshing.prime.SubtractZoneletsParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, retain_cutter: bool = None, extract_edges: bool = None, trace_edges: bool = None, json_data: dict = None, **kwargs)



   Parameters to use when subtracting zonelets.

   :param model: Model to create a ``SubtractZoneletsParams`` object with default parameters.
   :type model: Model
   :param retain_cutter: Retain the zonelets used for removal.
   :type retain_cutter: bool, optional
   :param extract_edges: Extract edges of intersection during subtract.
   :type extract_edges: bool, optional
   :param trace_edges: Trace edges of intersection on target. Only works if extractEdges is true.
   :type trace_edges: bool, optional
   :param json_data: JSON dictionary to create a ``SubtractZoneletsParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> subtract_zonelets_params = prime.SubtractZoneletsParams(model = model)

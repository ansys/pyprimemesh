:class:`MultiZoneEdgeBiasingParams`
===================================


.. py:class:: ansys.meshing.prime.MultiZoneEdgeBiasingParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, face_scope: ScopeDefinition = None, edge_scope: ScopeDefinition = None, bias_factor: float = None, n_divisions: int = None, json_data: dict = None, **kwargs)



   Defines MultiZone edge biasing control parameters.

   :param model: Model to create a ``MultiZoneEdgeBiasingParams`` object with default parameters.
   :type model: Model
   :param face_scope: Reference face zonelets to control mesh clustering orientation.

                      **This is a beta parameter**. **The behavior and name may change in the future**.
   :type face_scope: ScopeDefinition, optional
   :param edge_scope: Edge zonelets to control the expanse of edge biasing.

                      **This is a beta parameter**. **The behavior and name may change in the future**.
   :type edge_scope: ScopeDefinition, optional
   :param bias_factor: Bias factor used for MultiZone edge biasing control.

                       **This is a beta parameter**. **The behavior and name may change in the future**.
   :type bias_factor: float, optional
   :param n_divisions: Number of divisions on the section where edge biasing is done.

                       **This is a beta parameter**. **The behavior and name may change in the future**.
   :type n_divisions: int, optional
   :param json_data: JSON dictionary to create a ``MultiZoneEdgeBiasingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> multi_zone_edge_biasing_params = prime.MultiZoneEdgeBiasingParams(model = model)

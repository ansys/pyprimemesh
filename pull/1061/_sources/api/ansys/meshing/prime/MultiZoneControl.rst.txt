:class:`MultiZoneControl`
=========================


.. py:class:: ansys.meshing.prime.MultiZoneControl(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager, id: int, object_id: int, name: str)



   MultiZoneControl provides a way to gather all the information required for MultiZone meshing.

   The MultiZone meshing provides hex meshing capabilities.
   Different type of mesh can be generated using MultiZoneControl. For example, sweep mesh, map mesh and edge biased mesh.

   :param model: Server model to create MultiZoneControl object.
   :type model: Model
   :param id: Id of the MultiZoneControl.
   :type id: int
   :param object_id: Object id of the MultiZoneControl.
   :type object_id: int
   :param name: Name of the MultiZoneControl.
   :type name: str

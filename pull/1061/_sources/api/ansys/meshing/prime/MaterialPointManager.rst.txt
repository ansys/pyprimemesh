   Creates a material point at the given coordinates.


   :param suggested_name: Name suggested for the material point.
   :type suggested_name: str
   :param coords: Coordinates of material point.
   :type coords: Iterable[float]
   :param params: Parameters used to create material point.
   :type params: CreateMaterialPointParams

   :returns: Returns the result with material point name and id.
   :rtype: CreateMaterialPointResults

   .. rubric:: Notes

   A material point is created on calling this API.

   .. rubric:: Examples

   >>> material_point_results = model.control_data.create_material_point("Fluid", [1.0,2,0,3.0], params)

:class:`TopoFillHoleParams`
===========================


.. py:class:: ansys.meshing.prime.TopoFillHoleParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, edges_to_exclude: ansys.meshing.prime.params.primestructs.Iterable[int] = None, suppress_boundary_after_hole_fill: bool = None, fill_annular_hole: bool = None, json_data: dict = None, **kwargs)



   Parameters used to fill holes in topology.

   :param model: Model to create a ``TopoFillHoleParams`` object with default parameters.
   :type model: Model
   :param edges_to_exclude: TopoEdges to be excluded for cap creation.
   :type edges_to_exclude: Iterable[int], optional
   :param suppress_boundary_after_hole_fill: Option to preserve or suppress hole-boundary after filling holes.
   :type suppress_boundary_after_hole_fill: bool, optional
   :param fill_annular_hole: Option for filling holes with annular bounding loops.
   :type fill_annular_hole: bool, optional
   :param json_data: JSON dictionary to create a ``TopoFillHoleParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> topo_fill_hole_params = prime.TopoFillHoleParams(model = model)

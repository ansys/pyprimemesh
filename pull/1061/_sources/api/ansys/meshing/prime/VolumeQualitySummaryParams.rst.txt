   Parameters to control volume quality summary results.

   :param model: Model to create a ``VolumeQualitySummaryParams`` object with default parameters.
   :type model: Model
   :param cell_quality_measures: List of cell quality measures for volume quality diagnostics.
   :type cell_quality_measures: List[CellQualityMeasure], optional
   :param scope: Scope of the cell zonelets for volume quality diagnostics.
   :type scope: ScopeDefinition, optional
   :param quality_limit: Quality limit per cell quality measure. If the quality limit is not specified, the default quality limit is used.
   :type quality_limit: Iterable[float], optional
   :param json_data: JSON dictionary to create a ``VolumeQualitySummaryParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> volume_quality_summary_params = prime.VolumeQualitySummaryParams(model = model)

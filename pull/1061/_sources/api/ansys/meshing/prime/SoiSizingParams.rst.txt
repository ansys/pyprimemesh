:class:`SoiSizingParams`
========================


.. py:class:: ansys.meshing.prime.SoiSizingParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, centers: ansys.meshing.prime.params.primestructs.Iterable[float] = None, radius: float = None, max: float = None, growth_rate: float = None, json_data: dict = None, **kwargs)



   Parameters for sphere of influence size control.

   :param model: Model to create a ``SoiSizingParams`` object with default parameters.
   :type model: Model
   :param centers: Centers of the spheres within which the size is limited to the specified max value.

                   **This is a beta parameter**. **The behavior and name may change in the future**.
   :type centers: Iterable[float], optional
   :param radius: Radius of the sphere within which the size is limited to the specified max value.

                  **This is a beta parameter**. **The behavior and name may change in the future**.
   :type radius: float, optional
   :param max: Maximum size within the sphere.

               **This is a beta parameter**. **The behavior and name may change in the future**.
   :type max: float, optional
   :param growth_rate: The rate at which size grows outwards from the sphere boundary.

                       **This is a beta parameter**. **The behavior and name may change in the future**.
   :type growth_rate: float, optional
   :param json_data: JSON dictionary to create a ``SoiSizingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> soi_sizing_params = prime.SoiSizingParams(model = model)







:class:`JSONEncoder`
====================


.. py:class:: ansys.meshing.prime.relaxed_json.encoder.JSONEncoder(*, skipkeys=False, ensure_ascii=True, check_circular=True, allow_nan=True, sort_keys=False, separators=None, default=None)

   Class to encode Object to JSON format.

   :param skipkeys: Skip keys, by default False.
   :type skipkeys: bool, optional
   :param ensure_ascii: Ensure that we use ASCII characters, by default True.
   :type ensure_ascii: bool, optional
   :param check_circular: Check circular loops, by default True.
   :type check_circular: bool, optional
   :param allow_nan: Allow not number values, by default True.
   :type allow_nan: bool, optional
   :param sort_keys: Order the keys, by default False.
   :type sort_keys: bool, optional
   :param separators: String separators, by default None.
   :type separators: Any, optional
   :param default: Default error message, by default None.
   :type default: Any, optional

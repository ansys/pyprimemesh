





:class:`SocketCommunicator`
===========================


.. py:class:: ansys.meshing.prime.internals.socket_communicator.SocketCommunicator(ip: Optional[str] = None, port: Optional[int] = None, timeout: float = 10.0, credentials=None, **kwargs)

   Bases: :py:obj:`ansys.meshing.prime.internals.communicator.Communicator`


   Manages communication with the socket server.

   :param ip: IP address where the server is located. The default is ``None``.
   :type ip: Optional[str], optional
   :param port: Port where the server is deployed. The default is ``None``.
   :type port: Optional[int], optional
   :param timeout: Maximum time to wait for connection. The default is ``10.0``.
   :type timeout: float, optional
   :param credentials: Credentials for connecting to the server. The default is ``None``.
   :type credentials: Any, optional

   :raises ConnectionError: Could not connect to server.

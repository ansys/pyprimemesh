:class:`VolumeControlParams`
============================


.. py:class:: ansys.meshing.prime.VolumeControlParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, cell_zonelet_type: ansys.meshing.prime.params.primestructs.CellZoneletType = None, skip_hexcore: bool = None, json_data: dict = None, **kwargs)



   Volume control parameters are used to define the volume type in the volume control.

   :param model: Model to create a ``VolumeControlParams`` object with default parameters.
   :type model: Model
   :param cell_zonelet_type: Cell zonelet type is used to define the type of the associated volume.
   :type cell_zonelet_type: CellZoneletType, optional
   :param skip_hexcore: Check whether to skip hexahedral cells generation in the core for this volume or not. Applicable only for volumeFillType set to HexcoreTet or HexcorePoly in the AutoMeshParams structure.
   :type skip_hexcore: bool, optional
   :param json_data: JSON dictionary to create a ``VolumeControlParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> volume_control_params = prime.VolumeControlParams(model = model)

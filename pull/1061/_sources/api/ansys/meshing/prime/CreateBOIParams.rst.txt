:class:`CreateBOIParams`
========================


.. py:class:: ansys.meshing.prime.CreateBOIParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, boi_type: BOIType = None, perform_initial_wrap: bool = None, wrap_size: float = None, flow_dir: FlowDirection = None, side_scale: float = None, wake_scale: float = None, wake_levels: int = None, suggested_part_name: str = None, suggested_label_prefix: str = None, number_of_threads: int = None, json_data: dict = None, **kwargs)



   Parameters used for BOI surface creation in the given axis.

   :param model: Model to create a ``CreateBOIParams`` object with default parameters.
   :type model: Model
   :param boi_type: Type of BOI offsetting.
   :type boi_type: BOIType, optional
   :param perform_initial_wrap: Perform an initial wrap to create a BOI if BOI type is OFFSETSURFACE.
   :type perform_initial_wrap: bool, optional
   :param wrap_size: Set wrap size greater than the largest gap size in the input when performing_initial_wrap is true.
   :type wrap_size: float, optional
   :param flow_dir: Assigns the offset direction of inflation.
   :type flow_dir: FlowDirection, optional
   :param side_scale: BOI side scaling factor.
   :type side_scale: float, optional
   :param wake_scale: BOI flow direction scaling factor.
   :type wake_scale: float, optional
   :param wake_levels: BOI levels.
   :type wake_levels: int, optional
   :param suggested_part_name: Suggested part name for created BOI surfaces.
   :type suggested_part_name: str, optional
   :param suggested_label_prefix: Suggested label name for created BOI surfaces.
   :type suggested_label_prefix: str, optional
   :param number_of_threads: Number of threads for multithreading.
   :type number_of_threads: int, optional
   :param json_data: JSON dictionary to create a ``CreateBOIParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> create_boiparams = prime.CreateBOIParams(model = model)

   Parameters for meshing the solid spline.

   :param model: Model to create a ``RefineTetMeshParams`` object with default parameters.
   :type model: Model
   :param nisr: Interpolation elements in the local r-direction.

                **This is a beta parameter**. **The behavior and name may change in the future**.
   :type nisr: int, optional
   :param niss: Interpolation elements in the local s-direction.

                **This is a beta parameter**. **The behavior and name may change in the future**.
   :type niss: int, optional
   :param nist: Interpolation elements in the local t-direction.

                **This is a beta parameter**. **The behavior and name may change in the future**.
   :type nist: int, optional
   :param tolerance: Tolerance for boundary refinement.

                     **This is a beta parameter**. **The behavior and name may change in the future**.
   :type tolerance: float, optional
   :param json_data: JSON dictionary to create a ``RefineTetMeshParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> refine_tet_mesh_params = prime.RefineTetMeshParams(model = model)

:class:`TetParams`
==================


.. py:class:: ansys.meshing.prime.autogen.automeshstructs.TetParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, quadratic: bool = None, json_data: dict = None, **kwargs)



   Parameters to control tetrahedral mesh generation.

   :param model: Model to create a ``TetParams`` object with default parameters.
   :type model: Model
   :param quadratic: Option to generate quadratic tetrahedral mesh. It is not supported with parallel meshing. It is only supported with pure tetrahedral mesh.
   :type quadratic: bool, optional
   :param json_data: JSON dictionary to create a ``TetParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> tet_params = prime.TetParams(model = model)







:class:`AddThicknessParams`
===========================


.. py:class:: ansys.meshing.prime.autogen.surfaceutilitystructs.AddThicknessParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, thickness: float = None, reverse_face_normal: bool = None, suggested_part_name: str = None, fix_intersections: bool = None, json_data: dict = None, **kwargs)

   Bases: :py:obj:`ansys.meshing.prime.params.primestructs.CoreObject`


   Parameters to add thickness for a given plane.

   :param model: Model to create a ``AddThicknessParams`` object with default parameters.
   :type model: Model
   :param thickness: To assign the offset distance of inflation.
   :type thickness: float, optional
   :param reverse_face_normal: To assign the direction of inflation.
   :type reverse_face_normal: bool, optional
   :param suggested_part_name: Suggested part name for created patching surfaces.
   :type suggested_part_name: str, optional
   :param fix_intersections: Fix intersections in concave regions.
   :type fix_intersections: bool, optional
   :param json_data: JSON dictionary to create a ``AddThicknessParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> add_thickness_params = prime.AddThicknessParams(model = model)

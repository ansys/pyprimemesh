:class:`DeadRegion`
===================


.. py:class:: ansys.meshing.prime.autogen.wrapperstructs.DeadRegion(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, face_zonelet_ids: ansys.meshing.prime.params.primestructs.Iterable[int] = None, dead_material_points: ansys.meshing.prime.params.primestructs.List[str] = None, hole_size: float = None, json_data: dict = None, **kwargs)



   DeadRegion defines a dead region for patch flow regions operation.

   :param model: Model to create a ``DeadRegion`` object with default parameters.
   :type model: Model
   :param face_zonelet_ids: Face zonelets enclosing dead region.

                            **This is a beta parameter**. **The behavior and name may change in the future**.
   :type face_zonelet_ids: Iterable[int], optional
   :param dead_material_points: Material points to identify dead region.

                                **This is a beta parameter**. **The behavior and name may change in the future**.
   :type dead_material_points: List[str], optional
   :param hole_size: Maximum hole size used for patching.

                     **This is a beta parameter**. **The behavior and name may change in the future**.
   :type hole_size: float, optional
   :param json_data: JSON dictionary to create a ``DeadRegion`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> dead_region = prime.DeadRegion(model = model)







:class:`JoinParams`

.. py:class:: ansys.meshing.prime.autogen.connectstructs.JoinParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, tolerance: float = None, use_absolute_tolerance: bool = None, remesh: bool = None, match_angle: float = None, overlap_zone_name: str = None, json_data: dict = None, **kwargs)

   Bases: :py:obj:`ansys.meshing.prime.params.primestructs.CoreObject`


   Parameters used for join.

   :param model: Model to create a ``JoinParams`` object with default parameters.
   :type model: Model
   :param tolerance: Overlap tolerance between overlapping zonelets.
   :type tolerance: float, optional
   :param use_absolute_tolerance: Tolerance provided is absolute value.
   :type use_absolute_tolerance: bool, optional
   :param remesh: Remesh at overlap surface boundary.
   :type remesh: bool, optional
   :param match_angle: Match angle determines face pair inclination for overlap consideration.
   :type match_angle: float, optional
   :param overlap_zone_name: Zone id to be assigned to overlap zonelets belonging to different zones.
   :type overlap_zone_name: str, optional
   :param json_data: JSON dictionary to create a ``JoinParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> join_params = prime.JoinParams(model = model)

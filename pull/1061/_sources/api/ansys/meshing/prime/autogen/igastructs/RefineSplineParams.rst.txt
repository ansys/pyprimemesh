:class:`RefineSplineParams`
===========================


.. py:class:: ansys.meshing.prime.autogen.igastructs.RefineSplineParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, refine_flag_u: bool = None, refine_flag_v: bool = None, refine_flag_w: bool = None, spline_refinement_type: SplineRefinementType = None, json_data: dict = None, **kwargs)



   Spline refinement parameters.

   :param model: Model to create a ``RefineSplineParams`` object with default parameters.
   :type model: Model
   :param refine_flag_u: Indicates whether refinement is applied in u direction.
   :type refine_flag_u: bool, optional
   :param refine_flag_v: Indicates whether refinement is applied in v direction.
   :type refine_flag_v: bool, optional
   :param refine_flag_w: Indicates whether refinement is applied in w direction.
   :type refine_flag_w: bool, optional
   :param spline_refinement_type: Type of spline refinement. Currently, supports h-refinement and p-refinement.
   :type spline_refinement_type: SplineRefinementType, optional
   :param json_data: JSON dictionary to create a ``RefineSplineParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> refine_spline_params = prime.RefineSplineParams(model = model)

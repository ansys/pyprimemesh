:class:`SurfaceUtilities`
=========================


.. py:class:: ansys.meshing.prime.autogen.surfaceutilities.SurfaceUtilities(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager)



   Performs various general surface utilities algorithms. For example, copy zonelets, resolve surface intersections.

   :param model: Server model to create SurfaceUtilities object.
   :type model: Model

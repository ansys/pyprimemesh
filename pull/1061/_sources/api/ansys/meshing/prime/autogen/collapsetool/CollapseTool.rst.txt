.. py:method:: split_and_collapse_on_zonelets(part_id: int, face_zonelets: ansys.meshing.prime.params.primestructs.Iterable[int], register_id: int, split_params: ansys.meshing.prime.params.primestructs.SplitParams, collapse_params: ansys.meshing.prime.params.primestructs.CollapseParams) -> ansys.meshing.prime.params.primestructs.CollapseResults

   Split and collapse elements on face zonelets with the specified register id.


   :param part_id: Id of a part.
   :type part_id: int
   :param face_zonelets: Ids of face zonelets.
   :type face_zonelets: Iterable[int]
   :param register_id: Id used to identify face elements to split and collapse.
   :type register_id: int
   :param split_params: Parameters to split longest edge of face elements before collapse.
   :type split_params: SplitParams
   :param collapse_params: Parameters to collapse shortest edge of face elements.
   :type collapse_params: CollapseParams

   :returns: Returns the CollapseResults.
   :rtype: CollapseResults

   .. rubric:: Examples

   >>> collapse_tool = prime.CollapseTool(model = model)
   >>> results = collapse_tool.split_and_collapse_on_zonelets(
   >>>     part_id=part.id,
   >>>     face_zonelets=part.get_face_zonelets(),
   >>>     register_id=register_id,
   >>>     split_params=prime.SplitParams(model=model),
   >>>     collapse_params=prime.CollapseParams(model=model))

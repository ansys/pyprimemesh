:class:`LeakPreventionParams`
=============================


.. py:class:: ansys.meshing.prime.autogen.controlstructs.LeakPreventionParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, material_points: ansys.meshing.prime.params.primestructs.List[str] = None, scope: ScopeDefinition = None, max_hole_size: float = None, n_expansion_layers: int = None, json_data: dict = None, **kwargs)



   LeakPreventionParams defines leakage prevention control parameters for wrapper.

   :param model: Model to create a ``LeakPreventionParams`` object with default parameters.
   :type model: Model
   :param material_points: Material points used for leak prevention control.
   :type material_points: List[str], optional
   :param scope: Scope used for leak prevention control.
   :type scope: ScopeDefinition, optional
   :param max_hole_size: Maximum hole size to prevent leakage into region.
   :type max_hole_size: float, optional
   :param n_expansion_layers: Number of layers to expand leaking region.
   :type n_expansion_layers: int, optional
   :param json_data: JSON dictionary to create a ``LeakPreventionParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> leak_prevention_params = prime.LeakPreventionParams(model = model)

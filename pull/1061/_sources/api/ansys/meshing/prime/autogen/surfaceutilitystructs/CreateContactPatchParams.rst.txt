   Parameters used for contact patch creation in the given axis.

   :param model: Model to create a ``CreateContactPatchParams`` object with default parameters.
   :type model: Model
   :param contact_patch_axis: Assigns the contact patch direction.
   :type contact_patch_axis: ContactPatchAxis, optional
   :param offset_distance: Source offset distance value.
   :type offset_distance: float, optional
   :param grouping_tolerance: Tolerance distance value to group regions for contact patch creation.
   :type grouping_tolerance: float, optional
   :param suggested_part_name: Suggested part name for created contact patch surfaces.
   :type suggested_part_name: str, optional
   :param suggested_label_prefix: Suggested label name for created contact patch surfaces.
   :type suggested_label_prefix: str, optional
   :param json_data: JSON dictionary to create a ``CreateContactPatchParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> create_contact_patch_params = prime.CreateContactPatchParams(model = model)

   Parameters to control delete shadowed topofaces operation.

   :param model: Model to create a ``VolumetricScaffolderParams`` object with default parameters.
   :type model: Model
   :param absolute_dist_tol: Specify distance tolerance between overlapping faces.

   :type only_check_exact_overlaps: bool, optional
   :param json_data: JSON dictionary to create a ``VolumetricScaffolderParams`` object with provided parameters.
   :type json_data: dict, optional

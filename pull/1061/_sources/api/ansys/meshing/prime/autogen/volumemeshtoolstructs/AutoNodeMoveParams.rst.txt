:class:`AutoNodeMoveParams`
===========================


.. py:class:: ansys.meshing.prime.autogen.volumemeshtoolstructs.AutoNodeMoveParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, quality_measure: ansys.meshing.prime.params.primestructs.CellQualityMeasure = None, target_quality: float = None, dihedral_angle: float = None, n_iterations_per_node: int = None, restrict_boundary_nodes_along_surface: bool = None, n_attempts: int = None, json_data: dict = None, **kwargs)



   Parameters used to improve volume mesh by auto node move.

   :param model: Model to create a ``AutoNodeMoveParams`` object with default parameters.
   :type model: Model
   :param quality_measure: Specify cell quality measure to be used for volume mesh improvement. The default value for cell quality measure is skewness.
   :type quality_measure: CellQualityMeasure, optional
   :param target_quality: Specify target quality used for the mesh improvement based on specified quality measure.
   :type target_quality: float, optional
   :param dihedral_angle: Dihedral angle used to mantain features of boundary face zonelets.
   :type dihedral_angle: float, optional
   :param n_iterations_per_node: Number of iterations per node to be moved.
   :type n_iterations_per_node: int, optional
   :param restrict_boundary_nodes_along_surface: Option to restrict the movement of the boundary node to the plane containing the boundary faces sharing the boundary node.
   :type restrict_boundary_nodes_along_surface: bool, optional
   :param n_attempts: Number of attempts to improve specified quality measure by node movement.
   :type n_attempts: int, optional
   :param json_data: JSON dictionary to create a ``AutoNodeMoveParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> auto_node_move_params = prime.AutoNodeMoveParams(model = model)







:class:`Scaffolder`
===================


.. py:class:: ansys.meshing.prime.autogen.scaffolder.Scaffolder(model: ansys.meshing.prime.autogen.coreobject.CommunicationManager, part_id: int)

   Bases: :py:obj:`ansys.meshing.prime.autogen.coreobject.CoreObject`


   Scaffolder is used for achieving connections in structures made of sheets and beams. Solid bodies should be suppressed before applying scaffolding.

   :param model: Server model to create Scaffolder object.
   :type model: Model
   :param part_id: Id of the part.
   :type part_id: int

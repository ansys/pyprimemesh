:class:`BoundaryFittedSplineParams`
===================================


.. py:class:: ansys.meshing.prime.autogen.igastructs.BoundaryFittedSplineParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, degree_u: int = None, degree_v: int = None, degree_w: int = None, refinement_fraction_u: float = None, refinement_fraction_v: float = None, refinement_fraction_w: float = None, control_points_count_u: int = None, control_points_count_v: int = None, control_points_count_w: int = None, n_refine: int = None, control_point_selection_type: ControlPointSelection = None, json_data: dict = None, **kwargs)



   Boundary fitted spline fitting parameters.

   :param model: Model to create a ``BoundaryFittedSplineParams`` object with default parameters.
   :type model: Model
   :param degree_u: Degree of spline in u direction.
   :type degree_u: int, optional
   :param degree_v: Degree of spline in v direction.
   :type degree_v: int, optional
   :param degree_w: Degree of spline in w direction.
   :type degree_w: int, optional
   :param refinement_fraction_u: Fraction of input mesh size that sets the control points size in u direction. This is used in program controlled control points selection mode.
   :type refinement_fraction_u: float, optional
   :param refinement_fraction_v: Fraction of input mesh size that sets the control points size in v direction. This is used in program controlled control points selection mode.
   :type refinement_fraction_v: float, optional
   :param refinement_fraction_w: Fraction of input mesh size that sets the control points size in w direction. This is used in program controlled control points selection mode.
   :type refinement_fraction_w: float, optional
   :param control_points_count_u: Spline control points count in U direction. Used in manual control points selection mode.
   :type control_points_count_u: int, optional
   :param control_points_count_v: Spline control points count in V direction. Used in manual control points selection mode.
   :type control_points_count_v: int, optional
   :param control_points_count_w: Spline control points count in W direction. Used in manual control points selection mode.
   :type control_points_count_w: int, optional
   :param n_refine: Spline refinement level for rendering.
   :type n_refine: int, optional
   :param control_point_selection_type: Spline control points selection type.
   :type control_point_selection_type: ControlPointSelection, optional
   :param json_data: JSON dictionary to create a ``BoundaryFittedSplineParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> boundary_fitted_spline_params = prime.BoundaryFittedSplineParams(model = model)

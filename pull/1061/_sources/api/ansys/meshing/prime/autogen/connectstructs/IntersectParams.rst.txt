:class:`IntersectParams`
========================


.. py:class:: ansys.meshing.prime.autogen.connectstructs.IntersectParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, tolerance: float = None, use_absolute_tolerance: bool = None, remesh: bool = None, collapse_feature_angle: float = None, collapse_target_skewness: float = None, json_data: dict = None, **kwargs)



   Parameters used for intersection.

   :param model: Model to create a ``IntersectParams`` object with default parameters.
   :type model: Model
   :param tolerance: Intersection tolerance.
   :type tolerance: float, optional
   :param use_absolute_tolerance: True if tolerance provided is absolute value.
   :type use_absolute_tolerance: bool, optional
   :param remesh: Local remesh at the intersection.
   :type remesh: bool, optional
   :param collapse_feature_angle: Angle to preserve features while performing collapse in improve operation.
   :type collapse_feature_angle: float, optional
   :param collapse_target_skewness: Perform collapse on faces with skewness above the provided target skewness.
   :type collapse_target_skewness: float, optional
   :param json_data: JSON dictionary to create a ``IntersectParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> intersect_params = prime.IntersectParams(model = model)

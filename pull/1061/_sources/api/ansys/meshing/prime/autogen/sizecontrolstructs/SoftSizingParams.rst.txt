





:class:`SoftSizingParams`
=========================


.. py:class:: ansys.meshing.prime.autogen.sizecontrolstructs.SoftSizingParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, max: float = None, growth_rate: float = None, json_data: dict = None, **kwargs)

   Bases: :py:obj:`ansys.meshing.prime.params.primestructs.CoreObject`


   Size field computed using soft size control enables you to set the maximum size on the scoped zonelets .

   When the soft sizing is computed for edges or faces, the size will be affected by size computed using other controls.
   The minimum size on the zonelet will be determined based on the influence of other size controls,
   else a uniform size will be maintained.

   :param model: Model to create a ``SoftSizingParams`` object with default parameters.
   :type model: Model
   :param max: Maximum size used for computing edge and face size using soft size control.
   :type max: float, optional
   :param growth_rate: Growth rate used for transitioning from one element size to neighbor element size.
   :type growth_rate: float, optional
   :param json_data: JSON dictionary to create a ``SoftSizingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> soft_sizing_params = prime.SoftSizingParams(model = model)

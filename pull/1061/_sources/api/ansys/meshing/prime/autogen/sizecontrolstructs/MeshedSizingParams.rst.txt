:class:`MeshedSizingParams`
===========================


.. py:class:: ansys.meshing.prime.autogen.sizecontrolstructs.MeshedSizingParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, growth_rate: float = None, json_data: dict = None, **kwargs)



   Size field computed using meshed size control enables you to set the size based on existing sizes.

   :param model: Model to create a ``MeshedSizingParams`` object with default parameters.
   :type model: Model
   :param growth_rate: Growth rate used for transitioning from one element size to neighbor element size.
   :type growth_rate: float, optional
   :param json_data: JSON dictionary to create a ``MeshedSizingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> meshed_sizing_params = prime.MeshedSizingParams(model = model)

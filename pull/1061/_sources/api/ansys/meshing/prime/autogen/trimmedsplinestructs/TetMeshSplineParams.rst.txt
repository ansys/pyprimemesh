:class:`TetMeshSplineParams`
============================


.. py:class:: ansys.meshing.prime.autogen.trimmedsplinestructs.TetMeshSplineParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, feature_angle: float = None, mode: TrimmedSolidSplineCutMode = None, decimation_factor: float = None, json_data: dict = None, **kwargs)



   Parameters for meshing the solid spline.

   :param model: Model to create a ``TetMeshSplineParams`` object with default parameters.
   :type model: Model
   :param feature_angle: Feature angle used in meshing of the solid spline.

                         **This is a beta parameter**. **The behavior and name may change in the future**.
   :type feature_angle: float, optional
   :param mode: Cut mode to specify rule for mesh cell selection in the volume mesh.

                **This is a beta parameter**. **The behavior and name may change in the future**.
   :type mode: TrimmedSolidSplineCutMode, optional
   :param decimation_factor: Decimation factor used in meshing of the solid spline.

                             **This is a beta parameter**. **The behavior and name may change in the future**.
   :type decimation_factor: float, optional
   :param json_data: JSON dictionary to create a ``TetMeshSplineParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> tet_mesh_spline_params = prime.TetMeshSplineParams(model = model)

:class:`LocalSurferParams`
==========================


.. py:class:: ansys.meshing.prime.autogen.surferstructs.LocalSurferParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, min_angle: float = None, max_angle: float = None, size_field_type: ansys.meshing.prime.params.primestructs.SizeFieldType = None, min_size: float = None, max_size: float = None, growth_rate: float = None, constant_size: float = None, smooth_boundary: bool = None, n_rings: int = None, json_data: dict = None, **kwargs)



   Parameters to perform local surface remeshing.

   :param model: Model to create a ``LocalSurferParams`` object with default parameters.
   :type model: Model
   :param min_angle: Minimum feature angle limit used to identify and preserve features.
   :type min_angle: float, optional
   :param max_angle: Maximum feature angle limit used to identify and preserve features.
   :type max_angle: float, optional
   :param size_field_type: Size field type used to generate surface mesh.
   :type size_field_type: SizeFieldType, optional
   :param min_size: Minimum size to be used in sizing for the surfer.
   :type min_size: float, optional
   :param max_size: Maximum size to be used in sizing for the surfer.
   :type max_size: float, optional
   :param growth_rate: Growth rate to be used to propagate sizes.
   :type growth_rate: float, optional
   :param constant_size: Constant size to be used in case of constant size field.
   :type constant_size: float, optional
   :param smooth_boundary: Option to extend local selection to get smooth boundary of selected elements.
   :type smooth_boundary: bool, optional
   :param n_rings: Number of rings to extend the registered face selection for remeshing.
   :type n_rings: int, optional
   :param json_data: JSON dictionary to create a ``LocalSurferParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> local_surfer_params = prime.LocalSurferParams(model = model)

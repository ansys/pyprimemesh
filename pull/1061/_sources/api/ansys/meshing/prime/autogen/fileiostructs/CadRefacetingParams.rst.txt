:class:`CadRefacetingParams`
============================


.. py:class:: ansys.meshing.prime.autogen.fileiostructs.CadRefacetingParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, cad_faceter: CadFaceter = None, faceting_resolution: CadRefacetingResolution = None, custom_surface_deviation_tolerance: float = None, custom_normal_angle_tolerance: float = None, max_edge_size_limit: CadRefacetingMaxEdgeSizeLimit = None, max_edge_size: float = None, json_data: dict = None, **kwargs)



   Parameters to refacet CAD during import.

   :param model: Model to create a ``CadRefacetingParams`` object with default parameters.
   :type model: Model
   :param cad_faceter: Specify the available choices for faceter. The available option is Parasolid. (Note: ACIS faceter is being deprecated from 25R1).
   :type cad_faceter: CadFaceter, optional
   :param faceting_resolution: Set the faceting resolution.
   :type faceting_resolution: CadRefacetingResolution, optional
   :param custom_surface_deviation_tolerance: Set custom tolerance for surface deviation in specified length unit.
   :type custom_surface_deviation_tolerance: float, optional
   :param custom_normal_angle_tolerance: Set custom tolerance for normal angle in degree.
   :type custom_normal_angle_tolerance: float, optional
   :param max_edge_size_limit: Specify maximum edge size limit for faceting.
   :type max_edge_size_limit: CadRefacetingMaxEdgeSizeLimit, optional
   :param max_edge_size: Set maximum edge size of the facets.
   :type max_edge_size: float, optional
   :param json_data: JSON dictionary to create a ``CadRefacetingParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> cad_refaceting_params = prime.CadRefacetingParams(model = model)

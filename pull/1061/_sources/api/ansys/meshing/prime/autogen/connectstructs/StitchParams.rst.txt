:class:`StitchParams`
=====================


.. py:class:: ansys.meshing.prime.autogen.connectstructs.StitchParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, tolerance: float = None, use_absolute_tolerance: bool = None, remesh: bool = None, enable_multi_threading: bool = None, type: StitchType = None, json_data: dict = None, **kwargs)



   Parameters used for stitch operation.

   :param model: Model to create a ``StitchParams`` object with default parameters.
   :type model: Model
   :param tolerance: Distance tolerance for stitching boundaries.
   :type tolerance: float, optional
   :param use_absolute_tolerance: True if tolerance provided is absolute value.
   :type use_absolute_tolerance: bool, optional
   :param remesh: Remesh at stitch connection.
   :type remesh: bool, optional
   :param enable_multi_threading: Option to run stitch in parallel using multithread.
   :type enable_multi_threading: bool, optional
   :param type: Stitch type depending on nature of surface boundary edges to be stitched.
   :type type: StitchType, optional
   :param json_data: JSON dictionary to create a ``StitchParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> stitch_params = prime.StitchParams(model = model)

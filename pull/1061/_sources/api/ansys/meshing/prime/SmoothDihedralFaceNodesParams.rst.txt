:class:`SmoothDihedralFaceNodesParams`
======================================


.. py:class:: ansys.meshing.prime.SmoothDihedralFaceNodesParams(model: ansys.meshing.prime.params.primestructs.CommunicationManager = None, min_dihedral_angle: float = None, tolerance: float = None, type: SmoothType = None, json_data: dict = None, **kwargs)



   Parameters to smooth dihedral face nodes.

   :param model: Model to create a ``SmoothDihedralFaceNodesParams`` object with default parameters.
   :type model: Model
   :param min_dihedral_angle: Minimum angle to be used to identify dihedral faces.
   :type min_dihedral_angle: float, optional
   :param tolerance: Tolerance relative to local mesh size to control smooth movement of nodes.
   :type tolerance: float, optional
   :param type: Option to inflate neighbor nodes of dihedral face edges or smooth dihedral face edge nodes to improve dihedral angle.
   :type type: SmoothType, optional
   :param json_data: JSON dictionary to create a ``SmoothDihedralFaceNodesParams`` object with provided parameters.
   :type json_data: dict, optional

   .. rubric:: Examples

   >>> smooth_dihedral_face_nodes_params = prime.SmoothDihedralFaceNodesParams(model = model)

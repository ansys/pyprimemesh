   Parameters used to import fluent meshing mesh.

   :param model: Model to create a ``ImportFluentMeshingMeshParams`` object with default parameters.
   :type model: Model
   :param append: Option to append imported mesh to existing mesh instead of resetting model to imported mesh.
   :type append: bool, optional
   :param enable_multi_threading: Option to import multiple files in parallel using multithreading.
   :type enable_multi_threading: bool, optional

   :type json_data: dict, optional
